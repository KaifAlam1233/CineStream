# CineStream 🎬

A movie streaming web app where you can search and watch any movie for free.

## Features

- Search any movie by name
- Watch movies directly on the website
- Trending movies on homepage
- Movie details — rating, cast, plot, genre
- Fully responsive design

## Tech Stack

- React.js
- OMDB API (movie data)
- StreamIMDb (video player)
- CSS3

## Getting Started

### Prerequisites
- Node.js installed

### Installation

1. Clone the repo
   `git clone https://github.com/KaifAlam1233/CineStream.git`

2. Install dependencies
   `npm install`

3. Create a `.env` file in the root folder
   `REACT_APP_OMDB_KEY=your_api_key`

4. Start the app
   `npm start`
